# Connections

| Pin        | STM32      | Device  |
| ---------- |:----------:|:-------:|
| TX         | PA1        | USB-TTL |
| RX         | PA0        | USB-TTL |
| SCL        | PB13       |   IMU   |
| SDA        | PB15       |   IMU   |
| ADO        | PB14       |   IMU   |
| NCS        | PD0        |   IMU   |

* The most important things in driving SPI is CS
